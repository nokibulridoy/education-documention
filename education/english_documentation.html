1. How to Get Started.

How to Get Started: You will receive a login link (web link) from TMSS. You need to enter that link in the address bar of your web browser to open the Edu Desk login page. TMSS will provide you with a user email and password, which you will need to enter into the email and password fields on the Edu Desk login page to log in.

The login page is shown in the image below:

How to Log In to the Software

To log in to the Edu Desk software, first ensure that your computer is connected to the internet. Then, open an internet browser. The recommended browser for this software is Google Chrome.

You can also use any other browser. After opening the internet browser, enter the login link provided by TMSS into the browser's address bar.

After typing the login link, press the Enter key on your keyboard. Once you press it, a login form will appear to access the software.


The login form contains two text boxes: one for the User Email Address and the other for the Password. After entering the email and password in the respective fields, you need to click the login button. If the email and password are valid, you will be granted access to the software. If they are not valid, you will be redirected back to the login form.

After logging in, you will see a dashboard. On the left side of the dashboard, there is a menu. At the top menu, you will find options like Demo User and Logout. On the right side, there is a search bar, which you can use to search. Clicking on the Logout option will redirect you back to the login page.



From the TMSS-ICT option, you can go to My Profile, and if you want to log out, you can do so from there as well.

Instructions for Assigning User Access in the Software

User Access is the feature where a user or employee is granted various permissions based on their designation. That means they are given permissions on what they are allowed or not allowed to do. To assign permissions to a user, go to the left menu, hover over the RBAC option, and click on User Access.


After clicking, a user list will appear. Once the list is visible, click on the category of the employee to whom you want to assign user access and permissions.



Follow the steps below to create an Admin Module

To create an Admin Module, go to the RBAC option, hover over it, and click on User Access. Once the user list appears, click on the “Module” tab.



If you want to add a new Module, click on the Add New option.



After clicking Add New, a form will appear. Fill out the form with the required information and click the submit button below. If you don’t want to proceed, click the close option.



Follow the steps below to create an Admin Role

To create an Admin Role, go to the RBAC option, hover over it, and click on User Access. Once the user list appears, click on the “Admin Role” category.



If you want to add a new Admin Role, click on the “Create Role” option.



To create a new Admin Role, click on the Add New option. A form will appear. Fill out the form and click the submit button. If you don’t want to proceed, click on close.



After clicking, a form will appear with pre-filled information. You can modify the data as needed and click the “Update” or “Submit” button at the bottom to save the changes.



If you want to delete any Admin Role data, click the Delete button to remove it.



Follow the steps below to assign User Access

To assign user access, go to the RBAC option, hover over it, and click on “User”. A user list will appear.



To assign permissions to a user, click on Add New.



After clicking Add New, a form will appear. Enter the email of the user and select the role you want to assign, then click submit.



If you want to edit a user, click the edit button, update the information, and submit.



If you want to delete user access data, click the delete button to remove it.
